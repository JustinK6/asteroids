## Dependencies
- MSVC 2015
- Cmake
- Cinder

## Game
### Objective
Shoot down as many asteroids as possible while dodging asteroids.
### Controls
Key | Action
---| ------
W | Accelerate Ship
A | Rotate left
D | Rotate Right
F | Fire
R | Return to Menu (From game over screen)
S | Start game (From menu screen)
